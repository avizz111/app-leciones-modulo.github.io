¿Qué son las tablas?

Las tablas tal como en nuestro día a día son cuadros donde podemos poner información en filas y columnas

¿Qué se componen las tablas?

¿Cómo declarar una tabla?

Para poder declarar una tabla se debe usar la etiqueta <table> y dentro de estas se pondrán todas las etiquetas que la formaran.


Las primeras etiquetas que se ponen son las de filas <tr>, seguido de esto se ponen las etiquetas <th> o <td> las cuales se usan para celdas de títulos o celdas de datos respectivamente

<table>
        <tr>
            <th>Nombre</th>
            <th>Edad</th>
            <th>Ciudad</th>
        </tr>
        <tr>
            <td>Juan Pérez</td>
            <td>27</td>
            <td>León</td>
        </tr>
        </tr>
            <td>María López</td>
            <td>34</td>
            <td>Guadalajara</td>
        </tr>
        <tr>
            <td>Carlos García</td>
            <td>22</td>
            <td>Monterrey</td>
        </tr>
        <tr>
            <td>Sofia Martínez</td>
            <td>29</td>
            <td>Puebla</td>
        </tr>
</table>

¿Cuál etiqueta sirve para hacer filas?

Atributos de las tablas

Como cualquier etiqueta de HTML las de las tablas no tienen excepción y estas pueden ser para fines prácticos o de diseño las cuales son


  1. <table>
    • border: define el ancho del borde de la tabla
    • cellpadding: define el espacio entre el contenido de una celda y su borde
    • cellspacing: define el espacio entre las celdas
    • width: define el ancho de la tabla
    • summary: da un resumen para lectores de pantalla
  2. <tr>
    • align: alinea horizontalmente el contenido
    • bgcolor: define el color de la fila
    • valign: alinea el contenido de manera vertical
  3. <th>
    • colspan: combina varias columnas en una sola celda
    • rowspan: combina varias filas en una sola celda
    • scope: define el ámbito de la celda de encabezado
  4. <td>
    • colspan: combina columnas en una celda
    • rowspan: combina filas en una celda
    • headers: referencia a celdas de encabezado relacionadas con esta celda
